🔧 Configurer votre domaine sur GitHub Pages

1
Aller dans Settings → Pages Dans votre repo GitHub Mouv-
2
Ajouter « Custom domain » Entrez votre domaine gratuit (ex: mouv-transport.tk)
3
Mettre à jour les DNS Pointer vers 185.199.108.153 (GitHub IP) chez Freenom
4
Activer HTTPS ✅ Cocher "Enforce HTTPS" dans GitHub Pages Settings
